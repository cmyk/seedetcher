# SeedEtcher

SeedEtcher is a custom Raspberry Pi Zero firmware forked from [SeedHammer](https://github.com/seedhammer/seedhammer).  
It scans or inputs seed phrases / descriptors and outputs **PDF seed plates** for laser printing → toner transfer → acid etching.
